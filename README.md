# WebGL 3D 模型编辑器

## 网站简介
本项目是一个基于 WebGL 的在线 3D 模型编辑器，支持模型导入、变形、纹理编辑、光照调节以及实时渲染预览。页面采用响应式布局，可在桌面与移动端流畅使用。

## 页面与功能
共计 8 个页面：首页（模型库）、编辑工作台、属性面板、材质库、光照设置、渲染预览、导出设置以及关于我们。每个页面对应独立功能模块，通过统一的状态管理实现数据同步。

## 多 AI 协作与验收
开发过程中采用多 AI 协作模式：AI 负责代码生成、UI 布局建议、性能分析以及自动化测试用例编写。人工审核团队进行功能验收、交互可用性检查以及安全审计，确保每个迭代均满足需求。

## GitHub Pages 部署与访问(入口 index.html)
将仓库推送至 GitHub 后，在 Settings → Pages 中选择 main 分支的 /docs 文件夹作为源，系统会自动构建并提供静态站点。访问入口为 https://<用户名>.github.io/<仓库名>/index.html，打开即可使用完整编辑器功能。
